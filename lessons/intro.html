Claude Cowork คืออะไร?

Claude Cowork คือ desktop application ที่ให้คุณมอบหมายงานที่ซับซ้อนให้กับ Claude AI — แล้วรับผลงานที่สมบูรณ์พร้อมใช้ครับ ต่างจาก Chatbot ธรรมดาที่ต้องคุยโต้ตอบทีละขั้น Cowork สามารถทำงานอัตโนมัติหลายขั้นตอน วิเคราะห์ไฟล์หลายร้อยหน้า เชื่อมต่อกับ Google Drive, Gmail, Calendar และทำงานเป็นชั่วโมงโดยไม่ต้องดูแล

Chat vs Cowork vs Code

💬 Chat — ถามตอบ

คำถามเร็ว brainstorm งานเล็กๆ ต้องคุย AI ทีละขั้น

🏢 Cowork (❤️ คอร์สนี้)

งานใหญ่: อ่านไฟล์หลายชิ้น / หลายขั้นตอน / เชื่อมต่อแอปอื่น / ทำซ้ำอัตโนมัติ

💻 Code — เขียนโปรแกรม

นักพัฒนาเขียน debug deploy โค้ด — ไม่จำเป็นสำหรับคอร์สนี้

คุณจะได้เรียนอะไร?

  • 🧠 พื้นฐาน AI และ Generative AI — ตั้งแต่ต้นจนใช้งานได้จริง
  • 🎯 RTF Framework — สูตรสั่งงาน AI ให้ได้ผลลัพธ์ที่ต้องการ
  • 🏢 Claude Cowork — ติดตั้ง ตั้งค่า ใช้งานครบทุกฟีเจอร์
  • 🔌 Connectors — เชื่อมต่อ Google Drive, Gmail, Calendar
  • 🤖 Plugins & Skills — สร้าง AI assistant เฉพาะทาง
  • 🚀 Automation — วางระบบทำงานอัตโนมัติ
  • 🏥 Healthcare Use Cases — ตัวอย่างจริงจากเภสัชกรรมและการแพทย์

ต้องการอะไรบ้าง?

✅ Claude Desktop (macOS หรือ Windows x64)

✅ สมัครสมาชิก Claude Pro/Max/Team/Enterprise

✅ Google Account (สำหรับ Connector)

❌ ไม่จำเป็นต้องเขียนโค้ด

❌ ไม่จำเป็นต้องมีพื้นฐาน IT

❌ ไม่ต้องเตรียมตัวล่วงหน้า
