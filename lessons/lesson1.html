📱 บทที่ 1 — รู้จักและติดตั้ง Claude Cowork

พี่หมีพาไปรู้จัก Claude Cowork แบบละเอียด 🔧

Claude Cowork คือโหมดพิเศษใน Claude Desktop App ที่เปลี่ยน Claude จาก Chatbot ธรรมดาให้กลายเป็น AI Agent ที่ทำงานอัตโนมัติได้ ครับ

📊 Cowork vs Chat ปกติ — ความแตกต่างอยู่ที่ไหน?

ประเด็น Chat ปกติ Claude Cowork
🔄 วิธีการทำงาน ถามตอบทีละขั้น ต้องรอเรา ทำงานอัตโนมัติหลายขั้น
🤖 ความเป็น Agent ไม่ใช่ Agent เป็นแค่ Chatbot ✅ Agent จริง ทำงานอย่างอิสระ
📁 Access Files & Apps ไม่สามารถ ✅ ผ่าน Connectors (Gmail, Drive, etc)
⚙️ Automation ไม่มี ✅ ตั้งเวลาให้ทำงานอัตโนมัติ
🛠️ Customizable ไม่ได้ ✅ สร้าง Custom Skills & Plugins
💼 ใช้งานที่ดี ตอบ Q&A เขียนเล่น ๆ 🏢 งานเชิงธุรกิจ Healthcare

💻 ระบบที่ต้องการ (System Requirements)

🖥️ Computer

  • macOS 13+ หรือ Windows 10+ หรือ Linux
  • RAM ขั้นต่ำ 8GB (16GB แนะนำ)
  • Internet ต้องมี

🔑 Account

  • Claude Account (paid หรือ free ก็ได้)
  • Claude Desktop App (ดาวน์โหลด claude.ai)
  • Internet ทำงานดี

🎯 3 วิธีการทำงานกับ Claude Cowork

1️⃣ Command Mode (ที่ง่ายที่สุด)

ให้คำสั่งไป Claude เอง ทำงาน เสร็จ รายงานผล

📋 ตัวอย่างงาน Healthcare:
• "เรียงจดหมายตอบกลับจากเภสัชกรเมื่อวานให้หมอ เป็นภาษาไทยศาสตร์"
• "สรุปรายงานการศึกษาโครงการ มะเร็งตับ จากไฟล์ PDF นี้ เป็นตาราง 3 คอลัมน์"
• "ดูปัญหา Side Effect ของยา ใน 5 เรกคอร์ด และแนะนำ Alternative"

⏱️ เวลา: Instant — ผลลัพธ์ออกมาใน 10-30 วินาที

ใช้เมื่อ: งานเล็ก ๆ กลาง ๆ ต้องการผลลัพธ์เร็ว

2️⃣ Research & Analysis Mode (ลึกขึ้น)

ให้ Claude ไปค้นคว้า วิเคราะห์ หาคำตอบ ไม่ต้องให้อีก

📋 ตัวอย่างงาน Healthcare:
• "ค้นหา Drug Interactions ระหว่าง Simvastatin + Amlodipine แล้วหาข้อมูล Clinical Evidence"
• "ดูรายงานวิจัยเกี่ยวกับ HTN ใน Thailand ปี 2024-2025 แล้วเขียน Summary"
• "วิเคราะห์ผลลัพธ์การสอบ OSCE ของเภสัชกรจบใหม่ คัดแบ่งคน strong vs weak"

⏱️ เวลา: 30 วินาที - 2 นาที (ตามความซับซ้อน)

ใช้เมื่อ: ต้องการความลึก ต้อง Research หลายแหล่ง

3️⃣ File Processing Mode (เหมาะสำหรับ Batch)

ให้ทำงานกับหลายไฟล์พร้อมกัน รวม ตรวจสอบ สร้างรายงาน

📋 ตัวอย่างงาน Healthcare:
• "อ่าน 50 ไฟล์ Survey จากเภสัชกร แล้วสรุปความคิดเห็น เป็นหมวดหมู่"
• "ตรวจสอบ 100 ใบสั่งยา หาคนที่ใช้ยาผิด (Drug Error) แล้วทำรายงาน"
• "อ่าน 20 Patient Case files แล้วจำแนกปัญหา ใส่ลงสเปรดชีต"

⏱️ เวลา: 1-5 นาที (ตามจำนวนไฟล์)

ใช้เมื่อ: ต้องการทำงาน Bulk หรือ Batch Processing

📦 Installation Steps (ทีละขั้น)

Step 1: ดาวน์โหลด Claude Desktop

ไปที่ claude.ai → Download Desktop App → เลือก macOS/Windows/Linux ของคุณ

Step 2: ติดตั้งและสมัครสมาชิก

เปิด App → สมัครสมาชิก Claude (ถ้ายังไม่มี) → Login ด้วย Account

Step 3: เปิด Cowork Mode

ใน Settings → Toggle "Enable Cowork" → Restart App

Step 4: ลอง command แรก

พิมพ์ "Write a hello world script" → สั่ง Claude ทำ → ดูผล

🏋️ แบบฝึกหัด 1: ติดตั้ง Cowork และสั่งงาน

พี่หมีเชื่อว่าคุณติดตั้งได้แน่นอนครับ! 💪

ที่จะทำ:

  1. ติดตั้ง Claude Desktop App (ถ้ายังไม่มี)
  2. เปิด Cowork Mode
  3. สั่งงาน: "Create a simple HTML page about [your favorite drug] with 5 key points"
  4. รอให้เสร็จและดูผลลัพธ์
  5. ลองแก้ไข Prompt อีก 1 ครั้ง ดู Cowork ทำงานต่างกันไหม
